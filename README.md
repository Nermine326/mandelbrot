# Mandelbrot Set
A simple Python script to plot the Mandelbrot Set.

## What is the Mandelbrot Set?
From Wikipedia:
```
The Mandelbrot set is the set of complex numbers c for which the function f(z)=z^2+c does not diverge when iterated from z=0.
```
